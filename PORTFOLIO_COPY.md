# Portfolio Content

## Project Title

OpenClaw Agent Suite — AI Agent Evaluation & Debugging Platform

## Short Description

A full-stack AI agent evaluation platform for reviewing multi-step agent traces, building rubrics, detecting unsafe tool usage, simulating workflow automation, and exporting structured evaluation reports.

## Long Description

OpenClaw Agent Suite is a production-ready React + JavaScript + Express + MongoDB application designed around real AI agent evaluation workflows. It allows reviewers to paste JSON-based agent traces, detect prompt injection, unsafe actions, missing confirmations, wrong tool usage, privacy risks, API errors, and hallucination risk. The platform also includes a reusable rubric builder, a safe workflow simulator, JWT authentication, MongoDB persistence, and exportable JSON reports.

## Tech Stack

React, JavaScript, Vite, Node.js, Express.js, MongoDB, Mongoose, JWT, bcrypt, Zod, Helmet, REST APIs, Custom CSS

## Key Features

- JWT authentication and protected dashboard
- MongoDB-backed evaluations and rubrics
- Agent trace debugging and scoring
- Prompt injection and privacy-risk detection
- Dangerous action confirmation gate detection
- Reusable rubric builder
- Workflow simulator for order status, refunds, and support tickets
- Exportable JSON reports
- Server-side tests
- Deployment-ready Vercel/Render configuration

## CV Bullet Version

Built OpenClaw Agent Suite, a full-stack AI agent evaluation platform using React, JavaScript, Express, MongoDB, and JWT Auth to inspect agent traces, detect unsafe tool usage, write rubrics, simulate workflows, and export structured evaluation reports.

## Interview Explanation

I built this project because AI agents need more than a good final answer. They need to use tools correctly, handle errors, protect private data, ask for confirmation before sensitive actions, and follow instructions safely. The app gives reviewers a way to inspect the full agent trace, score it with rubrics, and identify issues such as prompt injection, hallucinated outputs, wrong API inputs, and unsafe actions.

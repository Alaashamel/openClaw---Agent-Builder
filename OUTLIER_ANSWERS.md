# Outlier / OpenClaw Application Answers

## Why are you interested in this project?

I am interested in this project because it combines software engineering, AI agents, workflow analysis, and evaluation. I enjoy working on systems where an AI model needs to follow instructions, use tools correctly, avoid unsafe behavior, and complete real-world tasks reliably.

I have experience with full-stack development, backend APIs, JavaScript, Python, and building structured applications. I am also interested in AI evaluation, rubrics, prompt safety, and debugging agent behavior.

## Describe your AI agents experience

I have been building and studying AI agent concepts such as multi-step workflows, tool usage, API-based automation, prompt evaluation, trace debugging, and safety checks. I understand that agents need to plan steps, call tools correctly, handle errors, protect user data, and produce reliable final outputs.

To demonstrate this, I built OpenClaw Agent Suite, a full-stack project for reviewing agent traces, writing rubrics, detecting unsafe actions, and simulating real workflow automation.

## Relevant skills

JavaScript, React, Node.js, Express, REST APIs, MongoDB, JWT Auth, Python basics, SQL basics, AI agent workflows, prompt injection awareness, rubric design, trace debugging, privacy checks, error handling, and technical writing.

## Project explanation

OpenClaw Agent Suite is a full-stack AI agent evaluation platform. It allows reviewers to paste agent traces, analyze multi-step workflows, detect unsafe actions, identify missing tool inputs, flag prompt injection attempts, and generate structured rubric-based reports. It also includes a workflow simulator that demonstrates safe agent behavior with confirmation gates and mock API tools.
